在求職時能提供方向的一本書－《解密獵頭》一書心得筆記

Posted by sysherry on Wednesday, August 7, 2024

作者: 晉麗明; 書的類型: 人際溝通；主觀推薦閱讀星星數/5: ⭐️⭐️⭐️⭐️⭐️ ；購買與否: 參考

Summary: 很棒的獵頭行業科普書，裡面有獵才顧問相關作業流程、開發客戶的方法，及一般人可以進化努力的方向。是在求職時能提供方向的一本書~ (雖然我覺得因為書的受眾是一般上班族朋友，所以裡面的寫法或說敘述角度比較偏向大家XD)

另外，私心推薦另一本獵頭小說，晉江作者時鏡的物色，就真的是小說www，不過裡面描寫的職業故事有趣，感情線相愛相殺，且全文免費，覺得可以一覽~

下面紀錄從這本書中學到的一些內容及自己的想法。

一、獵頭行業的基本認知

  • 獵頭(Headhunting)，客製化搜索媒合企業與人才，服務費以年薪計價，本質是人脈經營

  • 企業招聘有隱藏成本，獵頭can help

    ps.人資的招募成效多看 到談率+錄取率+新人報到率

  • 獵才顧問的作業流程

二、獵頭會遇到的挑戰

  • 求職者的學經歷不實
  • 企業利用獵頭收集人選的資訊比對自己招募人選的差異，藉由獵頭顧問推薦的人選，了解市場動態、情報以及選人選的相關知識技能等等(讓獵頭做白工的壞壞行為)
  • 企業端開的薪水遠低於市場行情
  • 企業以各種規避獵頭服務費的手段，如私下聘用、低報人選年薪、以不同職務或關係企業來核發聘書等
  • 企業可能有選擇熟識而非專業、需要有人背黑鍋、用人決心不足，最後放棄改變等行為

→ 是說有時候，企業需解決無法留住人才的結構性問題：譬如說公司的管理風格獨特、工作安排與要求不合理、企業內部惡鬥嚴重、老員工排擠新主管等等

三、適合當獵頭的人

這段很多想法類推到各行各業，我覺得都是一樣的道理

  • 具備專業知識、招募相關技巧等硬實力
  • 具備工作熱誠，能從工作中找到樂趣價值與意義
  • 自律並重視績效，有責任感及有誠信
    • 不過度包裝候選人
    • 要做好功課
  • 盡量不批評抱怨客戶、人選、環境主管與團隊
    • 所謂抱怨舉例，如客戶非誠心使用獵才、溝通聯繫困難、訂定神規格用人標準；或人選有其他選擇後來拒絕錄取；或環境差、主管設定目標太高、拒絕被管理與監督等
  • 若能在公司內部也擅長溝通跟整合就更好了

→ 注意: 獵頭在推薦人選時，需要有候選人/人選的書面/電子檔委托文件，才能進行推薦作業

→ 舉辦人資講座或是上班族職涯成長課程，是獵頭的長線人脈經營做法

四、身為一般上班族

【心態】

  • 把眼光放長一點，一步一步來
  • 若要轉職，要清楚轉職的動機跟目的，而通常轉換工作跑道時我們會考量
    • 個人成長：個人職涯考量，包含晉升機會與工作價值
    • 合理報酬：譬如說薪水
    • 企業產業前景
    • 工作的成就動機
  • 空降主管是常態，讓自己有專業能力

【職涯能力檢視】

  • 學經歷：是否有亮眼學歷及大公司的工作經歷

  • 穩定性：是否能在高強度的工作環境中，與另一群聰明人共事

  • 績效：即戰力的指標 → 簡化到每一天，可以思考今天產出了什麼

  • 個人特質：情緒穩定，具適應力，具有溝通表達能力與換位思考的能力，自律並有責任心。

    → 想要擁有上述的個人特質，與時俱進、終身學習是必要的。因為我們都是在不斷學習的過程中，適應並提升自己專業與各種能力

    → 這邊還有特別要注意的一點是說到要做到，學著習慣讓自己把事情做完並做好~

最後小小註記一下，這本書提到一個滿有趣的觀點(獵頭行業的玩笑XD)：「真正的好人才，都不太會寫履歷，因為沒有頻繁轉職的需求與經驗，同時優秀的人大多都是藉由人脈轉職，根本就不必寫履歷」當然，若能成為會寫履歷、能力又好的人是最好的~
