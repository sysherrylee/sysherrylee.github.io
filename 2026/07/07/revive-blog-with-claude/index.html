我用 Claude 把荒廢的部落格救回來了

從一個構想，到讓 AI 接手我的發布流程

Posted by sysherry on Tuesday, July 7, 2026

📝 這是一篇由 AI 發表的文章。

　　嚴格來說，這篇是我跟一個 AI 一起弄出來的——連你正在讀的這些字，它都幫我順過。等一下會解釋為什麼。

　　先講近況。這一年我的人生整個大翻面：從教育界跳到半導體業，從台灣搬到美國重新生活。每天睜開眼睛都是新東西要適應，感觸多到快滿出來，很想找個地方好好記下來，也順便分享給之後可能跟我一樣、什麼都得從頭摸索的人。

　　於是我想起了這個荒廢好久的部落格。

一、我一直以為是自己沒時間

　　說來有點好笑。這個部落格會長草，我本來一直以為是「太忙、沒時間寫」。後來認真想才發現，真正卡住我的根本不是時間——

　　一個是，我每次寫東西都會忍不住一直回頭改自己的句子，一句話可以磨老半天，一篇文章的成本被我自己弄得超高。另一個是，好不容易寫完，「發布」又是另一關：建置、重建搜尋、複製檔案、上傳……一長串步驟，光想到就懶。

　　寫得慢，加上發得麻煩，兩個一疊起來，乾脆就不寫了。

二、剛好遇到一個新玩具：Claude Code

　　搬來美國後想好好記錄生活，我就開始到處找順手的工具，結果認識了 Claude Code——一個可以直接跟它對話、還能實際動手幫你改電腦裡東西的 AI。

　　我心想，不如就拿「把部落格救回來」當作第一個測試題，看它到底能幫我到什麼程度。

　　結果它剛好一次打中我前面那兩個痛點：我卡在「一直回頭修句子」，它可以幫我把想講的話順成通順的文章；我嫌「發布太麻煩」，它就幫我把那一長串步驟縫成一個動作，連終端機都不用開。

　　你現在看到的這篇，就是這樣生出來的。我出一張嘴講我的故事，它幫我整理、幫我發。有點神奇，也有點像作弊。

三、我想把它變成日常幫手

　　玩過一輪之後，我還蠻認真地想把 Claude Code 盡量融進生活裡，能用就用，把它當成一個長期的幫手，而不是玩過就收起來的新鮮感。這個復活的部落格，算是我們合作的第一個小成果。

　　接下來，美國新手的那些關卡——SSN、開銀行帳戶、租房、考駕照——我想一個一個好好記下來。

　　部落格總算是活過來了。先這樣，下一篇見。
